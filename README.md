# COVID2019 HUB

Centro de operaciones entre hospitales y residencias y ayudantes en la crisis covid 2019.

URL Provisional:

[https://covid2019hub.herokuapp.com](https://covid2019hub.herokuapp.com)



### Install

Requirements: 

- node v12.9.1
- yarn

```
yarn
```

### Run

Frontend

```
yarn dev
```

Backend

```
node server
```
